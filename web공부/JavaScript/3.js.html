외부 자바스크립트 파일로 적용

외부에 작성된 자바스크립트 파일은 .js 확장자를 사용하여 저장한다.

< script src="/examples/media/example.js" >< /script >

외부 자바스크립트 파일을 사용하면 웹의 내용을 담당하는 HTML 코드로부터 웹의 동작을 구현하는 자바스크립트 코드를 분리하여, 가독성과 유지 보수가 쉽고 로딩 속도 또한 빨라진다.
