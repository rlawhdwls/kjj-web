문서정보meta

meta 요소는 head 요소안에 위치하며, 문서의 다양한 문서 정보를 나타낼 떄 사용한다.
meta 요소로 지정한 정보들은 화면에 표시되지 않지만, 기기가 해당 정보를 이해하고 분석할 수 있도록 정보(데이터)를 제공한다.
보통 name속성과 content속성을 함께 사용하며, name 속성에 값을 지정하고 지정된 값에 해당하는 정보를 content 속성에 명시한다.
만약 content 속성이 지정되지 않았다면 meta 요소는 지정되지 않은 것과 같이 해석될 수 있기 때문에 삭제한다.

    meta 요소에 사용할 수 있는 name 속성 값

  • description - 문서의 요약 설명문 제공
  • keyword - 문서의 키워드 지정
  • refresh - 페이지 새로 고침
  • author - 웹 문서 제작자
  • expires - 문서의 유효 기간 지정
  • pragma - 브라우저 문서 캐시 확인
  • robot - 검색 엔진 제어 여부
  • viewport - 뷰 포트 영역 조절

스타일 시트 삽입style

특정 HTML문서에만 CSS를 적용하고자 할 때 사용할 수 있는 요소로, HTML5에서는 scoped속성이 새롭게 추가 되었습니다.
scoped속성은 특정 문서 내 모든 콘텐츠가 아닌 특정 콘텐츠 블록에만 CSS를 적용할 수 있도록 합니다.

문서의 기준 URL 지정base

링크나 이미지 등과 같이 문서에 담겨 있는 모든 상대 URL의 기준이 되는 기본 URL을 지정할 수 있다.
head요소의 내부에 작성하며, 링크의 기준이 되어야 하기 때문에 URL이 사용된 요소들보다 먼저 나타나야 한다.
그리고 문서에 단 한 번만 작성되어야 한다.
기준이 2개가 되면 언제 어떤 것을 연결해야 할지 혼란스럽기 때문이다.
