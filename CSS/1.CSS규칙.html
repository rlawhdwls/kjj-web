CSS 기본 문법

CSS는 웹의 표현을 담당하는 언어로, 일반적인 컴퓨터 언어와는 다른 구조를 갖는다.
선택자(selector)와 선언부(declaration block)로 이루어진다.
선언부는 속성(property)과 속성 값(property value)으로 이루어지며, 선언부는 세미콜론(;)으로
속성과 속성 값을 구분하여 여러개의 선언(declaration)을 지정할 수 있다.
예) selector{property : value ; property : value}

CSS 주석 및 단위

CSS에서는 코드에 주선(comment)을 추가하고자 할 때 /* ㅁㄴㅇㄹ */로 할 수 있다.
CSS는 속성 값을 지정할 때 여러 형식과 단위를 사용한다.

  • 문자열 타입 - inherit와 같이 CSS에서 미리 정의된 키워드나 제작자가 정의한 저작자 키워드 등 텍스트로 입력하는 속성 값이 이에 속한다.
  • 숫자 타입 - 개수 비율들을 나타낼 수 있는 정수, 실수 값, 퍼센트 값이 이에 속한다.
  • 길이 단위 타입 - 길이 단위 타입은 상대 길이 단위 값과 절대 길이 단위 값이 있다. 상대 길이 단위 값은 특정 길이에 비율로 정해지는 값이다.

    상대 단위
  • em - 폰트의 기본 크기 값에 비례한 단위
  • ex - 폰트의 기본 X 높이에 비례한 단위
  • ch - '0'의 기본 폭에 비례한 단위
  • rem - 최상위 요소의 폰트 크기에 비례한 단위
  • vw - 뷰포트 너비에 비례한 단위
  • vh - 뷰포트 높이에 비례한 단위
  • vmin - 뷰포트의 최소 너비, 높이에 비례한 단위

    절대 단위
  • cm - 센티미터 단위
  • mm - 밀리미터 단위
  • in - 인치 단위, 2.54cm
  • px - 픽셀 단위, 1/96inch
  • pt - 포인트 단위, 1/72inch
  • pc - 피타 단위, 12pt

CSS 색상

CSS3에서의 색상 단위는 좀 더 확대 되었다. 특히, 투명값(alpha)이나 색상, 채도, 명도 등을 지정 할 수 있는 HSL 방식 그리고 currentcolor 키워드 등이 추가 되었다.

RGBA방식 - color : rgba(R, G, B, 투명도)
HSLA방식 - color : hsla(색상, 채도, 명도, 투명도)

CSS 적용하기

HTML 문서에 CSS를 적용하는 방법은 외부의 파일을 HTML 문서에 연결하는 External 방식,
HTML 문서 head부분에 style 요소를 선언하고 포함시키는 Embedded 방식,
HTML 문서의 특정 요소 내에 CSS 코드를 지정하는 Inline 방식으로 나눈다.

CSS 선택자

  • 전체 선택자 - *{}
  • 요소 선택자 - e{}
  • 클래스 선택자 - .class{}
  • 아이디 선택자 - #id{}
  • 속성 선택자 - a[attr]{}
  • 가상 클래스 선택자 - a:seu{}
  • 가상 요소 선택자 - p:seu{}
  • 선택자 결합

    선택자 결합은 문맥이나 요소의 구조를 기반으로 결합하여 사용하는 방법이다.
    종류로는 하위 선택자, 자식 선택자, 형제 선택자 등이 있다.

    • 하위 선택자 - #main div{}
    • 자식 선택자 - #main > div{}
    • 형재 선택자
      • + - h1+p{}
      • ~ - .note~p{}
    • 선택자 그룹화 - h1, h1+p, .note{}

  • 선택자 우선순위

    • 구체적일수록 높고, 포괄적일수록 낮다
    • id > class > tag
    • 마지막 > 처음
