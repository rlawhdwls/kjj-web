color

color는 요소의 전경색을 지정하는 속성이다. 앞에서 살펴본 바와 같이 색상과 관련된 값은 CSS3에서 기존 생강\ 값인 색의 3원색, 빨강, 초록, 파랑 값으로 표현하는 RGB형식뿐만 아니라 색상, 체도, 명도를 의미하는 HSL형식과 함께 투명도 등도 제공 된다.

연습 글자 - color(rgba)
연습 글자 - color(hsl)
연습 글자 - color(hsla)

opacity

opacity는 CSS3에 새롭게 추가된 속성으로 요소 박스의 투명도를 지정하는 속성이다. opacity 속성값은 0에서 1사이의 값을 갖고, 0은 투명 1은 불투명이다. 상속이된다.

opacity - 0.5
opacity - 0.3

background-color

배경색을 지정하는 속성으로, 범위는 기본적으로 content영역과 padding 영역이다.

연습글자 - rgba(255,0,0,0.5)

background-image

요소 박스에 배경 이미지를 지정하는 속성이다. CSS3에서는 배경 이미지 속성 값에 콤마(,)로 구분하여 여러 개의 이미지를 지정함으로써 하나의 요소에 다중 배경 이미지를 지정할 수 있다.

linear-gradient()

linear-gradient 지정 시 angle은 각도를 의미한다. deg 단위의 값이다 to[키워드] 또는 0%[백분율]와 같은 형식으로 지정할 수 있다. 키워드 top, bottom, left, right 로 to top right와 같이 상하 방향과 좌우 방향을 조합하여 사용할 수 있다.

color-stop은 선형 그레이디언트의 색상 변환 점을 지정하기 위한 옵션으로, 색상 값과 변환 점 위치 두가지 값을 콤마(,)로 구분하여 여러 개를 지정할 수 있다. 변환 점의 위치는 길이 단위 값이나 백분율 값으로 나타내며, 선형 그레이디언트 angle 방향의 위치이다. color-stop에서 변환 점의 위치 값은 생략할 수 있으며, 변환 점 값이 생략될 경우에는 색상 변환 점이 그레이디언트 angle 방향에 대하여 등 간격으로 배치된다.

연습글자 - linear-gradient
