11.인터랙션 요소와 전역 속성

상세설명 details, summary

details 요소는 추가 정보를 사용자에게 제공하고자 할 때 사용한다. details 요소는 summary 요소와 함께 사용하여 정보의 세부 사항을 마크업할 수 있습니다. 세부 사항은 논리 속성인 open 속성을 사용하여 처음부터 보이게 하거나 사용자의 선택에 의해 보여지도록 할 수 있다. 기존에는 이러한 처리를 모두 자바스크립트에 의존했지만, HTML5에 추가된 details 요소를 사용하면 자바 스크립트 없이도 이러한 구현이 가능하다.

전역속성

전역속성 설명

HTML5의 전역속성(Global Attributes)이란, 모든(HTML) 요소에서 사용할 수 있는 속성을 의미한다. HTML5에서는 기존에 제공되던 전용 속성 이외에 accesskey, contenteditable, draggable, hidden, spellcheck, data-* 속성등이 새롭게 추가되었다.

전역속성 값 의미 사용(예)
accesskey 문자 키보드 단축키를 지정 accesskey="s"
class 문자열 요소에 이름정의(중복된 class명 사용가능) class="note"
contenteditable 논리속성 사용자가 컨텐츠를 수정할 수 있는지를 지정 contenteditable
data-* 문자열 사용자 정의 속성을 추가 data-customer="customer1004"
dir ltr, rtl 문자열의 방향 지정 dir="rtl"
draggable true, false, auto 사용자가 컨텐츠를 드래그할 수 있는지를 지정 draggable="true"
hidden 논리 속성 컨텐츠를 숨길 때 사용 hidden
id 문자열 요소에 이름 정의(중복된 id명 사용 불가) id="gnb"
spellcheck true, false 스펠링 체크 기능의 적용 여부를 지정할 때 사용 spellcheck="true"
style 문자열 특정 요소에 인라인 스타일을 지정할 때 사용 style="color:#000;"
tabindex 숫자 탭 키로 이동 시 이동 순서를 지정할 때 사용 tabindex="1"
title 문자열 컨텐츠에 대체 설명을 지정할 때 사용 title="자세한 설명은 하단의 내용을 참조하세요"
