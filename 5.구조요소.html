콘텐츠 섹션section

일반적으로 콘텐츠를 장이나 절 등으로 구분하고자 할 경우에 사용한다.
장이나 절 등으로 구분 되기 때문에 section요소 안에는 헤딩 관련 요소를 사용하여
해당 섹션의 제목을 지정해야 합니다. 제목이 없는 경우에는 장이나 절의 섹션이라고 할 수 없기 때문에
반드시 제목을 제공해야 한다.
  1. 레이아웃을 구현하기 위해 기존에 사용하던 div요소를 section요소로 대치하여 사용하는 경우
    section요소의 잘못된 사용으로 논리적인 구조보다는 보여지는 디자인을 중심으로 마크업하기 때문에 발생한다.
  2. section요소는 레이아웃 관점에서 열이나 행을 구분하기 위해 사용하면 안된다.
  3. 제목을 가진 섹션의 하위 섹션으로 구성되면 제목과 본문을 다른 섹션으로 해석한다.
  4. 컨텐츠의 구성을 하나의 섹션으로 마크업하고 배치를 위해 각 단락을 구분하고자 할 경우 section요소가 아닌
    div 요소를 사용해야한다.

섹션요소 사용

섹션 요소의 컨텐츠 모델

section 요소는 섹셔닝 컨텐츠에 속한 요소로 아웃라인에 영향을 준다.

내비게이션nav

nav요소는 내비게이션을 마크업할 때 사용한다. 내비게이션은 메인메뉴나 서브 메뉴 등을 의미하며,
서로 다른 페이지로 이동할 수 있는 항목을 포함한다.
